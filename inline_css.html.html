Understanding Generic Font Families

Generic font families include serif, san-serif, monospace, cursive, and fantastic. Each represents a group of fonts with similar characteristics. For example, serif fonts have small lines at the ends of characters, while sans-serif fonts do not.

Ways to Indicate color in css

Colors in css can be defined using color names (like " red"), hexadecimal values (like "#FF000"), RGB values (like "rgb(255,0,0)"), HSL value, and even transparent vlues uisng RGBA or HSLA.
